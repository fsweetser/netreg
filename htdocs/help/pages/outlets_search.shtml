Searching Outlets

If you have a large number of outlets registered, or if you are a member of a group that has access to a large number of outlet registrations, you may want to search for a particular outlet to view or edit rather than wade through the list to find it.

To search for machines matching some criteria:

  1. Click on the [Search Outlets] link at the top of the Registered Outlets table on the Main page.

    The Search Outlets page is displayed.

  2. Enter your search criteria in the fields shown. Note that only outlets matching ALL of the criteria you enter will be found.

    The following fields are available:

    Type
    Cable
    Device
    Port
    Attributes
    Flags
    Status
    User/Group/Affiliation

  3. Click on the Search button.

    The outlets matching all of the criteria you entered will be displayed.

  4. Click on an ID in the Search Results page to view/update information for that outlet.

    If no outlets, or too many outlets, are listed in the search results, use your browser's Back button to return to the Search Outlets page and enter new criteria.
