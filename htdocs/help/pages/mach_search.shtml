Searching Machines

If you have a large number of machines registered, or if you are a member of a group that has access to a large number of machine registrations, you may want to search for a particular machine to view or edit rather than wade through the list to find it.

To search for machines matching some criteria:

  1. Click on the [Search Machines] link at the top of the Registered Machines table on the Main page.

    The Search Machines page is displayed.

  2. Enter your search criteria in the fields shown. Note that only machines matching ALL of the criteria you enter will be found.

    The following fields are available:

    Hardware Address
    IP Address
    Hostname
    Mode
    Subnet
    Domain
    IP Address Zone
    User/Group/Affiliation

  3. Click on the Search button.

    The machines matching all of the criteria you entered will be displayed.

  4. Click on a hostname in the Search Results page to view/update information for that machine.

    If no machines, or too many machines, are listed in the search results, use your browser's Back button to return to the Search Machines page and enter new criteria.
