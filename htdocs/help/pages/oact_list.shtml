Outlet Activations

When a user requests that an outlet be activatedand the system determines that the outlet is not pre-activated, it is placed in a queue based on the building that it is in. Each technician is assigned one or more buildings that they are responsible for. The "Outlet Activation" page is used to access the contents of these queues.

Outlet Activation Query

To determine the activation queue to look at, you can select either to look at a single building, or all buildings that a particular technician is responsible for.

Clicking on the "Activation Queues" link will take you to the page where you can edit the definitions of the technician queues.

Pending Activations

This section shows a list of all pending activation requests that match the query selected in the Outlet Activation Query section.

By clicking on the outlet number, you will be taken to the "Outlet Information" page.

To complete an activation, you must select the type of network service that is being activated on the outlet, check the "On" box, and fill in the device and port information in the appropriate blanks. To commit the changer, click on the "Update" button. NOTE: Be sure to use the SNMP port number, not the physical port number when filling in this form.

List of Activation Queues

These queues are normally associated with a technician. The queue name corresponds to the name of the technician. Clicking on the name of a queue will allow you to edit or delete the queue. To add a new queue, click on "[Add Activation Queue]." If there are a large number of queues, you can navigate among them using the "[First Page]", "[Next Page]", "[Last Page]" and the numbered links.

Editing Queue Information

The name of a queue can be changed by typing the new name in the "Queue Name" field and then clicking "Update." This is useful when a new technician takes over for one leaving the group and you wish to transfer all building responsibilities.

To delete the entire queue, click on "[Delete Queue]." This will leave all buildings in the queue unassigned. To add another building to the queue, select it from the pull-down menu at the bottom of the page and click "Add Building". To delete a single building from the queue, click on "Delete" on the line corresponding to the building you want to remove.

Clicking on a building will display the buildi information for that building.
